# BlockSentinel

BlockSentinel is a lightweight Ethereum wallet monitoring and risk analysis tool.

## Features

- Track wallet transactions
- Basic risk scoring
- JSON reports
- Easy API integration

## Setup

1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Copy .env.example to .env and add your API keys
4. Run:
   python main.py
